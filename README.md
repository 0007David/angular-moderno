# List of content 

Perfecto — lo que tienes es un temario Angular clásico (v2–v15 aprox.), centrado en:

Modules
Services tradicionales
RxJS-heavy
Two-way binding clásico


Pero en [Angular 16 → 21](../angular.moderno.md) el enfoque cambió bastante hacia:

Standalone APIs
Signals (reactividad moderna)
Zoneless
Control Flow nuevo (@if, @for)
Mejor performance y DX
